##CSS声明
CSS声明总是以分号结束，声明总以大括号括起来
```css
p {color:red;text-align:center;}
```
为了增强可读性，你可以每行只描述一个属性
```css
p
{
    color:red;
    text-align:center;
}
```
##CSS注释
css注释是以\/*开始，以\*/结束
```css
/*这是个注释*/
p
{
    text-align:center;
    /*这是另一个注释*/
    color:black;
    font-family:arial;
}
```
